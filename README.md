# Learn OpenGL

A minimal LearnOpenGL starter project using GLFW and GLAD, built on Void Linux with a simple Makefile.

## Install Dependencies

Install the required development packages (Void Linux):

```bash
sudo xbps-install -S MesaLib-devel glfw-devel libXrandr-devel libXi-devel libX11-devel
```

## Build

```bash
make
```

## Run

```bash
./app
```
